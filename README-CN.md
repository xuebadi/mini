# 小宇宙 Builder - Tiny World Builder 中文版

这是一个基于 [Tiny World Builder](https://github.com/jasonkneen/tiny-world-builder) 的中文版本。

## 特性

- ✅ **完全中文化**：界面默认显示中文
- ✅ **体素世界编辑**：在 8×8 网格上建造你的小世界
- ✅ **智能拼接系统**：相邻元素会自动连接（路径、河流、栅栏、房屋等）
- ✅ **17种元素**：地形、建筑、装饰、生物
- ✅ **天气系统**：动态云朵、天气变化、时间流转
- ✅ **多种视角**：等距、俯视、透视、第一/第三人称
- ✅ **本地保存**：支持多槽位保存
- ✅ **零依赖**：双击 HTML 文件即可运行，无需安装

## 如何开始

### 方法 1：本地打开（推荐）
1. 双击 `tiny-world-builder.html` 文件
2. 浏览器会自动打开应用
3. 开始建造你的小世界！

### 方法 2：部署到 Web
1. 将整个 `xiaoyuzhou` 文件夹上传到任意静态托管服务
   - Vercel
   - Netlify
   - GitHub Pages
   - 任何支持静态文件的服务
2. 访问你的部署地址

## 操作指南

### 基础操作
- **放置元素**：点击工具栏选择工具，然后在网格上点击
- **删除元素**：按 `E` 键或选择橡皮擦工具
- **旋转视角**：鼠标拖拽
- **缩放**：鼠标滚轮或双指捏合
- **平移**：右键拖拽或空格+拖拽

### 高级操作
- **抬高/降低地形**：按 `R` 键抬高，`F` 键降低
- **堆叠**：同一格子可以堆叠相同物体，最多 8 层
- **切换视角**：按 `P` 或 `I` 键切换相机模式
- **命令面板**：按 `Ctrl+P` 快速切换工具和设置

### 智能拼接示例
- 🛤️ **路径**：相邻路径会自动连接成完整道路
- 🌊 **河流**：水域边缘会自动生成河岸过渡
- 🏠 **房屋**：相邻房屋会根据排列自动变成 L 型、T 型等建筑群
- 🪨 **岩石**：多块岩石会形成崎岖的山石群

## 技术栈

- **纯原生 JS**：约 1600 行代码，无框架依赖
- **Three.js r128**：3D 渲染引擎（已自托管）
- **单文件架构**：核心逻辑集中，易于理解和修改
- **离线可用**：所有资源本地化，无需网络

## 文件结构

```
xiaoyuzhou/
├── tiny-world-builder.html    # 主文件（默认中文）
├── engine/                    # 核心引擎
│   ├── i18n/                 # 国际化文件
│   │   ├── zh.js            # 中文翻译
│   │   ├── en.js            # 英文翻译
│   │   └── i18n-core.js     # 国际化核心（已修改默认语言）
│   └── world/               # 世界逻辑
├── styles/                   # 样式文件
├── assets/                   # 资源文件
├── vendor/                   # 第三方库（Three.js）
└── models/                   # 3D 模型
```

## 语言切换

虽然默认是中文，但你仍然可以切换到其他语言：

1. 点击界面右上角的语言按钮（🇨🇳）
2. 选择你想要的语言：
   - English (英文)
   - Français (法语)
   - Español (西班牙语)
   - 中文 (简体中文)
   - ไทย (泰语)

## 保存和分享

- **本地保存**：点击"世界菜单" → "另存为新世界"
- **导出 JSON**：点击顶部工具栏的"导出 JSON"按钮
- **导入 JSON**：点击"导入 JSON"按钮加载之前保存的世界
- **分享链接**：点击"复制分享链接"生成可分享的 URL

## 常见问题

### Q: 为什么打开后界面是英文？
A: 检查浏览器是否缓存了之前的语言设置。尝试清除 localStorage：
```javascript
localStorage.removeItem('tinyworld:lang');
location.reload();
```

### Q: 能否离线使用？
A: 可以！所有资源都已本地化，包括 Three.js。直接双击 HTML 文件即可。

### Q: 如何修改默认世界大小？
A: 编辑 `engine/world/29-persistence-api.js` 中的网格配置。

### Q: 能否添加自定义元素？
A: 可以！参考 `engine/world/19-tools-toolbar.js` 添加新工具。
